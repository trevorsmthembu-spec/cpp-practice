# Reverse A Given String

## Problem

Write a program that prints a string in reverse order.

## Concepts Practiced

- Functions
- Strings
- for loops
- String indexing

## Example

**Input**

```text
Reverse a Given String.
```

**Output**

```text
.gnirtS neviG a esreveR
```

## Time Complexity

- **Time:** O(n)
- **Space:** O(1)
